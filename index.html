Tic Tac Toe

Basic two player TIC-TAC-TOE game board
A basic two player tic-tac-toe gameboard developed as Project Zero while pursuing Web Development - Immersive course at General Assembly.

Tech Stack
  • jQuery
  • sweetalert
  • javaScript
  • HTML5/CSS3

    • Features
    The game is played on a grid that has 9 squares (3x3). The choice is between player one i.e.. "X" and second player "O". Players take turns putting their marks on empty squares. The first player to get 3 marks in a row (up, down, across, or diagonal) is the winner.

    Casual Hire

    A twist to AirTasker. A simple job search platform matching job posters and job searchers.

    This project was built in a week by a team of three including myself. We built a job search portal similar to AirTasker. The exciting bits in this project are the use of Geocoder and Google Maps API to plot jobs nearby, category based search using Isotope and use of materialized views. This project was built as a part of Web Development - Immersive course at General Assembly.

    Second Opinion

    A platform for patients who primarily want to seek a second opinion on their present medical condition with a specialist.

    This is an individual project made as a part of General Assembly's Web Development Immersive Course. Considering the amount of time it takes to get a specialist appointment, I tried to create a platform for patients who primarily want to seek a second opinion on their present medical condition with a specialist. The exciting parts of this project are - Payment Gateway using Stripe, Speciality Based Search, Doctor - Patient Conversation/Messaging Platform This project was made within a week.
